恢复

注意：如果用的是windows备份的，最好恢复也是在windows平台，用其他平台也可以恢复，可能会有部分依赖安装失败问题。

这里用windows平台举例，假设windows机器刚装完系统，未配置任何环境，先安装 git、nodejs 环境。

克隆远程仓库，注意，拉取 source 分支即可

git clone -b source 仓库名称

进入该仓库目录，安装依赖：

npm install
