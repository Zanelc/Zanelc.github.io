HTML学习笔记

基础认知

Web标准的构成

三部分

构成 语言 说明
结构 HTML 页面元素和内容
表现 CSS 网页元素的外观和位置等页面样式（如：颜色、大小等）
行为 JAVASCRIPT 网页模型的定义与页面交互

HTML页面固定结构

  • html标签：网页的整体
  • head标签：网页的头部
  • body标签：网页的身体
  • title标签：网页的标题

注释

vscode中对应注释快捷键为：ctrl+/

<!-- 这是网页的注释-->

标签

  • 双标签：<>内容</> 前部分叫开始标签，后部分叫结束标签，两部分之间包裹内容

  • 单标签：<> 自成一体，无法包裹内容

Tips：当需要规定开始和结束时候，使用双标签；否则使用单标签。

  • 关系

    • 父子关系

      <head>
	<title></title>
</head>

    • 兄弟关系

      <head></head>
<body></body>

  • 常见标签

标签 说明
<h1></h1> 标题标签，共6级，从1到6重要性逐次递减，独占一行
<p></p> 段落标签，用于分段显示
<br> 换行标签，让文字强制换行
<hr> 水平线标签
<b>、<strong> 加粗标签
<u>、<ins> 下划线标签
<i>、<em> 倾斜标签
<s>、<del> 删除线标签

  • 路径

    • 绝对路径：盘符开头的地址、完整的网络地址
    • 相对路径：当前文件开始出发找目标文件的过程

  • 媒体标签

    1. <img> 图片标签

      • src：图片位置
      • alt：替换文本（图片加载失败时候显示）
      • title：提示文本（鼠标悬停图片上方显示）
      • height/width：单独设置一个，图片会等比例缩放，不会被拉伸

    2. <audio> 音频标签

      • src

      • controls：显示播放的控件

      • autoplay：自动播放（部分浏览器不支持）

      • loop：循环播放

    3. <video> 视频标签

      • src
      • controls
      • autoplay
      • loop

  • 超链接标签 <a>

    • href：跳转地址，开发网站初期，不知道跳转地址时候，href的值书写#
    • target：目标网页的打开形式
      1. _self：当前窗口跳转
      2. _blank：新窗口跳转

  • 列表标签

    1. 无序列表：ul列表中只能包含li标签，li标签可以包含任何内容

      <ul>
  <li>榴莲</li>
  <li>苹果</li>
  <li>香蕉</li>
</ul>

    2. 有序列表：ol、li

      <ol>
  <li>苹果</li>
  <li>华为</li>
  <li>小米</li>
</ol>

    3. 自定义列表：dl、dt、dd

      <dl>
  <dt>帮助中心</dt>
  <dd>购物管理</dd>
  <dd>账户管理</dd>
  <dd>订单操作</dd>
</dl>

  • 表格标签

    1. table：表格整体，用于包裹多个tr

    2. tr：表格每行

    3. td：表格单元格

      • colspan：跨列合并
      • rowspan：跨行合并

    4. 相关属性

      • border

      • width/height

    5. th：表头单元格

    6. caption：表格大标题，放在table标签内部

​ 嵌套关系：table > tr > td

  • 表单标签

    1. input：根据type属性值的不同，展示不同的效果

      type属性 说明 常用属性
      text 文本框 placeholder：占位符，提示字
      password 密码框 同上
      radio 单选框 name：分组，拥有相同name属性值的单选框为一组；checked：默认选中
      checkbox 多选框 checked
      file 文件选择，用于上传文件 multiple：上传多个文件
      submit 提交按钮 /
      reset 重置按钮 /
      button 普通按钮 value：按钮提示文字

    2. button

      • type属性值
        • submit
        • reset
        • button

    3. select：下拉菜单标签

      • select：下拉菜单的整体
      • option：下拉菜单的每一项
        • selected：默认选中

    4. textarea：文本域标签

      • cols：规定了文本域内可见宽度
      • rows：规定了文本域内可见行数

    5. label

      用lable标签吧内容和表单标签一起包裹起来，删除label标签的for属性即可

  • 无语义的标签

    1. div：一行只能显示一个
    2. span：一行可以显示多个

  • 有语义的布局标签–手机端常用

    场景：在HTML5新版本中，推出了一些有语义的布局标签

    标签名 语义
    header 网页头部
    nav 网页导航
    footer 网页底部
    aside 网页侧边栏
    section 网页区块
    article 网页文章

    注意：以上标签显示特点和div一致，但是比div多了不同的语义

字符实体

  • 常见字符实体

    • 场景：在网页中展示特殊符号效果时，需要使用字符实体替代

    • 结构：&英文;

    • 常见字符实体

      显示结果 描述 实体名称
      空格 &nbsp;
      < 小于号 &lt;
      > 大于号 &gt;
      & 和号 &amp;
      “ 引号 &quot;
      ‘ 撇号 &apos;
