CSS学习笔记

基础认识

  • css（层叠样式表）作用

    给页面中的HTML标签设置样式

  • 写在哪里？

    写在style标签中，style标签一般写在head标签里面，title标签下面

  • 引入方式

    1. 内嵌式：CSS写在style标签中
    2. 外联式：CSS写在一个单独的css文件中（通过link标签在网页中引入）
    3. 行内式：CSS写在标签的style属性中

基础选择器

  • 标签选择器
    1. 结构：标签名{CSS属性名：属性值；}
    2. 标签选择器选择的是一类标签，而不是单独的某一个
  • 类选择器
    1. 结构：**.类名{CSS属性名：属性值；}**
    2. 所有标签上都有class属性，class属性的属性值称为类名（类似于名字）
    3. 一个标签可以同时有多个类名，类名之间以空格隔开
    4. 类名可以重复，一个类选择器可以同时选中多个标签
  • id选择器
    1. 结构：**#id属性值{CSS属性名：属性值；}**
    2. id属性值类似于身份证号码，在一个页面中是唯一的，不可重复的！
    3. 一个标签只能有一个id属性值，一个id选择器只能选中一个标签
  • 通配符选择器
    1. 结构：***{CSS属性名：属性值；}**
    2. 找到页面中所有的标签，设置样式

字体和文本样式

  • 字体样式
    1. 字体大小：font-size 取值：数字+px
    2. 字体粗细：font-weight
      • 取值：关键字：normal 、bold ；纯数字：正常（400）、加粗（700）
    3. 字体样式（是否倾斜）：font-style
      • 取值：normal（正常）；italic（倾斜）
    4. 字体系列：font-family
      • 无衬线字体（sans-serif）：粗细均匀，首尾无装饰
      • 衬线字体（serif）：粗细不均，首尾有笔锋装饰
      • 等宽字体（monospace）：宽度相等
  • 样式的层叠问题
    • 如果给一个标签设置了相同的属性，后面的样式会覆盖前面的
  • 字体font相关属性的连写
    1. 属性名：font（复合属性）
    2. 取值：font：style weight size family
  • 文本样式
    1. 文本缩进：text-indent
      • 取值：数字+px；数字+em（1em=当前标签的font-size大小）
    2. 文本水平对齐方式：text-align
      • 取值：left、center、right
    3. 文本修饰：text-decoration
      • 取值：underline（下划线）、line-through（删除线）、overline（上划线）、none（无装饰线，常用于清除a标签默认的下划线）
  • 行高
    1. 作用：控制一行的上下行间距
    2. 属性名：line-height
    3. 取值
      • 数字+px
      • 倍数（当前标签font-size的倍数）
  • 颜色常见取值
    • 属性名
      • 文字：color
      • 背景颜色：background-color
  • 标签水平居中
    • margin：0 auto

选择器进阶

  • 复合选择器

    1. 后代选择器：空格
      • 作用：根据HTML标签的嵌套关系，选择父元素后代中满足条件的元素
      • 语法：选择器1 选择器2{css}
    2. 子代选择器：**>**
      • 作用：根据HTML标签的嵌套关系，选择父元素子代中满足条件的元素
      • 语法：选择器1 > 选择器2{css}

  • 并集选择器：，

    1. 作用：同时选择多组标签，设置相同的样式
    2. 语法：选择器1,选择器2{css}

  • 交集选择器：紧挨着

    1. 作用：选中页面中同时满足多个选择器的标签
    2. 语法：选择器1选择器2{css}

  • hover伪类选择器

    1. 作用：选中鼠标悬停在元素上的状态，设置样式
    2. 语法：选择器:hover{css}

  • emmet语法

    1. 作用：通过简写语法，快速生成代码

    2. 语法

    记忆 示例 效果
    标签名 div <div></div>
    类选择器 .red <div class=”red”></div>
    id选择器 #one <div id=”one”></div>
    交集选择器 p.red#one <p class=”red” id=”one”><p>
    子代选择器 ul>li <ul><li></li><ul>
    内部文本 ul>li{内容} <ul><li>内容</li><ul>
    创建多个 ul>li*3 <ul><li></li><li></li><li></li><ul>

背景相关属性

  • 背景颜色
    1. 属性名：background-color
    2. 属性值：关键字、rgb表示、rgba表示、十六进制…
    3. 背景颜色默认值是透明
  • 背景图片
    1. 属性名：background-image
    2. 属性值：url(‘图片的路径’);
  • 背景平铺
    1. 属性名：background-repeat
    2. 属性值
      • repeat：水平和垂直方向都平铺
      • no-repeat：不平铺
      • repeat-x：水平方向平铺
      • repeat-y：垂直方向平铺
  • 背景位置
    1. 属性名：background-position
    2. 属性值
    3. 方位名词
      • 水平方向：left、center、right
      • 垂直方向：top、center、bottom
    4. 数字+px
      • 盒子的左上角（0,0）原点
      • x轴 水平向右
      • y轴 水平向下
  • 背景相关属性连写
    1. 属性名：background
    2. 书写顺序：color image repeat position
  • img标签和背景图片的区别
    1. img标签是一个标签，不设置宽高默认以原尺寸显示
    2. div标签+背景图片，背景图片只是装饰的CSS样式，不能撑开div标签

元素的显示模式

  • 块级元素

    1. 特点
    2. 独占一行
    3. 宽度默认是父元素宽度，高度默认由内容撑开
    4. 可以设置宽高
    5. 代表标签：div、p、h系列、ul、li、dl、dt、dd、form、header…

  • 行内元素

    1. 特点

    2. 一行可以显示多个

    3. 宽度和高度默认由内容撑开

    4. 不可以设置宽高

    5. 代表标签：a、span、b、u、i、s…

  • 行内块元素

    1. 特点
    2. 一行可以显示多个
    3. 可以设置宽高
    4. 代表标签：input、textarea、button、select…

  • 元素显示模式转换

    • 语法

      属性 效果 使用频率
      display：block 转换成块级元素 较多
      display：inline-block 转换成行内块元素 较多
      display：inline 转换成行内元素 极少

CSS特性

  • 继承性
    1. 特点：子元素有默认继承父元素样式的特点
    2. 可以继承的常见属性(文字控制属性都可以继承)：
      • color
      • font-style、font-weight、font-size、font-family
      • text-indent、text-align
      • line-height
      • …
  • 层叠性
  • 优先级
    1. 特性：不同的选择器有不同的优先级，优先级高的选择器样式会覆盖优先级低选择器样式
    2. 优先级公式：继承<通配符选择器<标签选择器<类选择器<id选择器<行内样式<!important

盒子模型

  • 盒子模型
    1. 内容区域（content）
    2. 内边距区域（padding）
    3. 边框区域（border）
    4. 外边距区域（margin）
  • 内容的宽度和高度
    1. 作用：利用width和height属性默认设置是盒子内容区域的大小
    2. 常见取值：数字+px
  • 边框（border）-连写形式
    1. 属性值：单个取值的连写，取值之间以空格隔开；如：border: 10px solid red;
    2. 单方向设置：border-方位名词
  • box-sizing: border-box;
    • 变成CSS3的盒子模型，好处：加了border和padding不需要手动减法
  • 清楚默认内外边距：margin: 0;padding: 0;

CSS浮动

  • 结构伪类选择器

    1. 作用：根据元素在HTML中的结构关系查找元素

    2. 选择器

      选择器 说明
      E:first-child{} 匹配父元素中的第一个子元素，并且是E元素
      E:last-child{} 匹配父元素中的最后一个子元素，并且是E元素
      E:nth-child(n){} 匹配父元素中的第n个子元素，并且是E元素
      E:nth-last-child(n){} 匹配父元素中的倒数第n个子元素，并且是E元素

    3. n的注意点

      • n为：0、1、2、3、4、5…

      • 通过n可以组成常见公式

        功能 公式
        偶数 2n、even
        奇数 2n+1、2n-1、odd
        找到前5个 -n+5
        找到第5个往后 n+5

  • 伪元素：由CSS模拟出标签效果，一般页面中的非主体内容可以使用伪元素
