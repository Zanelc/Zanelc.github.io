Javascript学习笔记

基础语法

js简介

  • javascript介绍
    1. js是一种运行在客户端(浏览器)的编程语言，实现人机交互效果。HTML+CSS不属于编程语言
    2. 作用
      • 网页特效
      • 表单验证
      • 数据交互(获取后台的数据，渲染到前端)
      • 服务端编程(node.js)
    3. 组成
      • ECMAScript：规定了js基础语法核心知识
      • Web APIs
        • DOM操作文档，比如对页面元素进行移动、大小、添加删除
        • BOM操作浏览器，比如页面弹窗，检测窗口宽度、存储数据到浏览器
    4. 书写位置
      • 内部Javascript：直接写在html文件里，用script标签包住；script标签写在上面
      • 外部Javascript：代码写在以.js结尾的文件里 <script src="my.js"></script>
      • 内联Javascript：代码直接写在标签内部 <button onclick="alert('内联')">b</button>
    5. 注释
      • 单行注释：//
      • 块注释：/* */
    6. 结束符
      • 使用英文的;代表语句结束
      • 实际开发中，可写可不写，浏览器(Javascript引擎)可以自动推断出语句的结束位置
    7. 输入和输出语法
      • 输出语法
        • document.write(“要出的内容”)
        • alert(“要出的内容”)
        • console.log(“控制台打印”)
      • 输出语法
        • prompt(“请输入你的名字：”)

变量

  • 变量是什么：变量是计算机存储数据的 “容器”

  • 变量的基本使用

    1. 声明

      • 语法：let 变量名

    2. 赋值

      • 赋值符号：**=**

    3. 更新变量

      • 变量赋值后，可以通过简单的给它一个不同的值来更新它,let不允许多次声明一个变量

        let age=18
age=18

    4. 变量本质

      • 是程序在内存中申请的一块用来存放数据的小空间

  • let 和 var 的区别：在较旧的JavaScript中，使用关键字var来声明变量，而不是let

    1. var声明
      • 可以先使用，再声明(不合理)
      • var声明过的变量可以重复声明(不合理)
      • 比如变量提升、全局变量、没有块级作用域等等
    2. var已经废弃，统一使用let

  • 数组：let 数组名=[数据1、数据2、数据3] 获取数组的长度，数组.length

常量

  • 基本使用

    1. 概念：使用const声明的变量称为“常量”
    2. 使用场景：当某个变量永远不会改变的时候，就可以使用const来声明，而不是let
    3. 注意：常量不允许重新赋值，声明的时候必须赋值(初始化)

  • 数据类型：弱数据类型语言

    1. 基本数据类型
      • (number)数字型 ：NaN(Not a Number)
      • (string)字符串型 ： 通过单引号(‘’)、双引号(“”)或反引号(``)包裹的数据都叫字符串
        • 模板字符串：外边用反引号(``)，里面用 ${}
      • (boolean)布尔型
      • (undefined)未定义型：表示没有赋值
      • (null)空类型：表示赋值了，但是内容为空
    2. 引用数据类型
      • object对象
    3. 通过typeof关键字检测数据类型
    4. 类型转换：把一种数据类型的变量转换为我们需要的数据类型
      • 隐式转换
        • +号两边只要有一个是字符串，都会把另一个转成字符串
        • +号作为正号解析可以转成数字型console.log(+'123')
      • 显式转换
        • Number(数据)
        • parseint(数据)：只保留整数
        • parseFloat(数据)：可以保留小数

  • 运算符

    1. 赋值运算符
    2. 一元运算符：++i：先自加；i++：先运算；独立使用时候，两者没有区别
    3. 比较运算符：==：左右两边值是否相等；===：左右两边是否类型和值都相等
    4. 逻辑运算符：&& 、||、 ！
    5. 运算符优先级

  • 语句

    1. 表达式是可以求值的代码，语句是一段可以执行的代码

    2. 分支语句

      • IF分支语句

        if(条件){
	满足条件执行的代码
}
else if(条件){
    代码
}
else{
    不满足条件执行的代码
}

      • 三元运算符

        条件?满足条件执行的代码:不满足条件执行的代码

      • switch语句

        switch(数据){
	case 值1:
		代码1
		break
	case 值2:
		代码2
		break
	...
	default:
		代码n
		break
}

    3. 循环语句

      • while循环

        while(循环条件){
	要重复执行的代码(循环体)
}

      • 循环退出

        • break：退出循环
        • continue：结束本次循环，继续下次循环
